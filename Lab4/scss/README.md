# Лабораторная работа 4 — SCSS / SASS / Less

## Установка

```
npm install
```

Установятся локально пакеты `sass` и `less` (см. `package.json`).

## Компиляция в реальном времени (задание 1)

```
npm run watch
```

Команда отслеживает изменения и пересобирает CSS-файлы для заданий
2, 3 (SCSS) и 4 (SASS). При любом сохранении исходного файла рядом
с ним обновляется `styles.css`.

## Однократная сборка (включая Less)

```
npm run build
```

## Структура

| Папка | Что внутри |
| --- | --- |
| `task2/` | Задание 2 — список из 100 чисел, цвет `rgb(n, 256-n, 0)` через `@for` |
| `task3/` | Задание 3 — макет страницы (миксины + вложенные правила, SCSS) |
| `task4-sass/task2/` | Задание 4 — задание 2 на синтаксисе SASS (отступы) |
| `task4-sass/task3/` | Задание 4 — задание 3 на синтаксисе SASS |
| `task5-less/task2/` | Задание 5 — задание 2 на Less |
| `task5-less/task3/` | Задание 5 — задание 3 на Less |

Открывать в браузере следует `index.html` внутри каждой папки.
