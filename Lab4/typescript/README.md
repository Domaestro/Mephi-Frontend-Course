# Лабораторная работа 4 — TypeScript

## Установка

```
npm install
```

Установится локальный `typescript` (см. `package.json`).

## Компиляция в реальном времени (задание 1)

```
npm run watch
```

Команда запускает `tsc --watch` и при любом изменении файла внутри
`src/` пересобирает соответствующий `.js` в папке `dist/`.

## Однократная сборка

```
npm run build
```

## Запуск

```
npm run run:task2
npm run run:task3
...
```

## Структура

| Файл | Что внутри |
| --- | --- |
| `src/task1.ts` | Задание 1 — простой пример для проверки компиляции |
| `src/task2.ts` | Задание 2 — класс `User` с интерфейсом `IUser` |
| `src/task3.ts` | Задание 3 — тот же класс, но через псевдоним типа `TUser` |
| `src/task4.ts` | Задание 4 — перегруженная функция `distance` |
| `src/task5.ts` | Задание 5 — бинарное дерево поиска (без `as` и `!`) |
| `src/task6.ts` | Задание 6 — паттерны Adapter, Strategy, Observer |

В `tsconfig.json` включены `strict: true` и `noImplicitAny: true`,
как этого требует задание 5.
